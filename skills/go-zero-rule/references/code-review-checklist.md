# 代码审查清单

## 提交前检查

在提交代码前，请确认以下各项：

- [ ] 代码已通过 `gofmt` 和 `goimports` 格式化
- [ ] Handler 层仅做请求/响应处理，无业务逻辑
- [ ] 业务逻辑在 Logic 层实现
- [ ] 错误使用了自定义错误码，而非直接返回
- [ ] 所有日志使用 `WithContext` 保持链路追踪
- [ ] 敏感信息已脱敏处理
- [ ] 配置参数已放入 `etc/*.yaml`，无硬编码
- [ ] 单元测试覆盖率不低于 80%
- [ ] 事务操作使用 `TransactCtx` 方法
- [ ] API/Proto 定义文件与实际代码同步
- [ ] 中间件正确处理了错误和 panic
- [ ] 使用 goctl 生成代码，而非手写
