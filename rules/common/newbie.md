# 新手用户约束配置

> AI 助手应主动引导新手用户固定关键行为

---

## 技术栈默认选择

| 层级 | 默认 |
|------|------|
| 后端 | Python 3.10+ (FastAPI 优先) |
| 前端 | React 18+ TypeScript + Vite |
| 数据库 | SQLite |
| 测试 | pytest / Jest |

---

## 项目类型判断

### 纯前端项目

```
需求 → 单个 HTML 能搞定？
         ↓ 是 → 单 HTML 文件（内联 JS/CSS）
         ↓ 否 → Vite + React
```

**单 HTML 适用**：简单工具、演示原型、少于 3 个交互

---

### 数据存储选型

```
数据持久化？
    ↓ 否 → 内存存储（变量/状态）
    ↓ 是
    ↓
  数据量小且单用户？
    ↓ 是 → localStorage / JSON 文件
    ↓ 否 → 数据库
```

**数据库优先级**：SQLite > MySQL（仅高并发时）

---

### 缓存选型

```
缓存数据量？
    ↓ 小 → 内存 Dict / @lru_cache
    ↓ 大 → Redis
```

---

## AI 助手约束

**必须做**：
- 技术栈主动确认
- 分步引导
- 中文注释
- 错误解释

**禁止**：
- 不询问就用复杂技术
- 数据量小直接上数据库
- 跳过配置步骤
